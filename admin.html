🚀 Quick Start Guide
Step 1 — Set up Firebase
Edit firebase.js and paste your Firebase config. Enable Email/Password and Google auth in your Firebase console under Authentication → Sign-in methods.
Step 2 — Apply Firestore Rules
Open firebase.js, scroll to the bottom, copy the security rules, and paste them into Firestore → Rules in your Firebase console.
Step 3 — Make yourself admin
After registering on the site, go to Firestore → users → your UID document and change the role field from "student" to "admin".
Step 4 — Create a test
Click "Create Test" to fill in test metadata. Then click "Add Questions" to add questions one by one, or use "JSON Upload" to bulk-upload from a file.
Step 5 — Publish
In "Manage Tests", toggle a test to Published so students can see it on the homepage.

📖 JSON Format Reference

      
{
  "questions": [
    {
      "order": 1,
      "section": 0,
      "sectionSub": "A",
      "type": "mcq",
      "text": "A particle executes SHM with amplitude <em>A</em>...",
      "options": ["Option A", "Option B", "Option C", "Option D"],
      "correctAns": 0,
      "posMarks": 4,
      "negMarks": -1
    },
    {
      "order": 21,
      "section": 0,
      "sectionSub": "B",
      "type": "integer",
      "text": "The moment of inertia of a solid sphere...",
      "options": [],
      "correctAns": "5.33",
      "posMarks": 4,
      "negMarks": 0
    }
  ]
}

      

        Fields:
        order = question number (1–90) ·
        section = 0 (Physics) / 1 (Chem) / 2 (Maths) ·
        sectionSub = "A" (MCQ) or "B" (Numerical) ·
        type = "mcq" or "integer" ·
        correctAns = index (0–3) for MCQ, string value for integer ·
        HTML tags allowed in text and options
